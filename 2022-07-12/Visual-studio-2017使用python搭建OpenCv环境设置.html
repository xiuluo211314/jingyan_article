方法/步骤 
1. 新建python应用程序，如图【文件】->【新建项目】->【Python应用程序】。Vs中安装Python教程参考经验如下：Visual Studio如何安装python运行环境？====================================================================


2. 新建.py脚本如下（选中项目名鼠标右键，菜单中选择【添加】->新建项->空Python文件）

 
3. 步骤2后输入代码：（准备test.jpg到d盘的路径下，和"D:\python\test.jpg"替换成自己的）
 #导入cv模块
import cv2 as cv
#读取图像，支持 bmp、jpg、png、tiff 等常用格式
img = cv.imread("D:\python\test.jpg")
#创建窗口并显示图像
cv.namedWindow("Image")
cv.imshow("Image",img)
cv.waitKey(0)
#释放窗口
cv2.destroyAllWindows()
 
4. ctrl + F5运行，出现如图所示的情况，说明OpenCv库没有引用进来
5. 打开命令行窗口，输入命令：pip,出现如图所示说明已经安装pip成功，否则的话参考经验如下：
https://jingyan.baidu.com/article/647f0115f45f4c7f2148a8b8.html
6. 输入命令如下： pip install opencv-python
点击回车运行程序，等待安装成功。
7. 输入命令：pip install numpy Matplotlib，等待安装成功。
