方法/步骤 
1.  打开Eclipse的软件，点击顶部菜单栏的【help】->【install new software...】

2. 在弹出的【install】窗口中的【work with】下拉菜单中找到你的eclipse软件对应的版本号，例如我的版本是Neon.

3. 稍等片刻，等待加载插件完成，勾选如图所示的【Web, XML, Java EE and OSGi Enterprise Development】

4. 步骤3后，点击底部的【Next】
5. 弹出的【Install Detials】窗口中，点击右下角的【Next】
6. 弹出的【Review Licenses】窗口中，勾选右下角的【I accept ...】，在点击Finish.
7. 安装完成后，需要重启软件如图所示，选择【Yes】
8. 创建web项目如图所示，新建工程，找到web->dynamic Web Project.
