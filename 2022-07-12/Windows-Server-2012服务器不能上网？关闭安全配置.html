Windows Server 2012服务器不能上网？关闭安全配置

标题
Windows Server 2012服务器不能上网？关闭安全配置
简介

Windows Server 2012启用或关闭Internet Explorer增强的安全配置?简单几步，让你轻松解决。
经验视频

工具/原料
联网电脑
方法/步骤 
1. 打开服务器，找到IE浏览器的图标，点击打开
 
2. IE10 会出现【已启用 Internet Explorer 增强的安全配置】的提示信息，如图所示
 
3.点击右上角的设置按钮，弹出的菜单中选择【Internet选项】
 
4.弹出窗口如图所示，打开安全标签页，其中安全等级中会设置为【高安全性】
 
5. 点击服务器图标，弹出的页面中点击【本地服务器】
6. 在属性面板中，点击IE增强的安全配置后的启用
7. 弹出的窗口中，分别勾选管理员和用户的关闭

8. 重启浏览器后，输入百度网页如图所示，即可以正常上网
