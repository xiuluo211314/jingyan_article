方法/步骤 
1. 使用pycharm软件打开工程，点击底部【Terminal】->【输入】pip命令后回车，如果输出下图所示的内容说明安装成功
 
2. 或者打开命令提示符，输入pip回车后，输出如图所示的命令，显示如图所示说明安装成功
 
3. 在命令行中输入命令： pip install validators 后回车，安装validators插件，如图所示说明安装成功
 
4. 插件的用法代码如下。如果输入的是url，输出的结果为true，否则为false
import validators
url = 'YOUR URL'
validators.url(url)
 
5. 测试邮箱大代码如下，显示结果为true
>>> import validators
>>> validators.email('someone@example.com')
True
6. 将代码加入到自己的工程中去，进行调试显示结果如图所示
