方法/步骤 
1. 键入网站下载和自己软件匹配的geckodriver（没有梯子的话，参考步骤3中的下载链接）
2. 找到火狐安装的文件路径的exe文件，如图所示
 
3. 链接: https://pan.baidu.com/s/1estRuUHF_8C4C96cHm6ODQ 提取码: q2xx 。下载文件后解压，将exe文件复制到步骤2找到的路径
 
4.推荐使用相匹配的历史版本：http://ftp.mozilla.org/pub/firefox/releases/

5.步骤5后，进入下载链接，进行下载并安装。安装完成先不要打开，先参考经验设置：firefox火狐浏览器如何取消自动更新？否则会自动更新，导致不匹配
6. 设置好环境变量，记得重启下IDE（PyCharm）.环境变量的路径指向步骤3中的geckodriver.exe

7. 打开IDE(pycharm)创建py文件输入代码如下，运行即可。
from selenium import webdriver # 导入webdriver包
import os

# dr = webdriver.Firefox() # 初始化一个火狐浏览器实例：driver
# dr=webdriver.Chrome(r"C:\Program Files (x86)\Google\Chrome\Application\chromedriver.exe") #调用谷歌浏览器
# firefoxBin = os.path.abspath(r"C:\Program Files\Mozilla Firefox\geckodriver.exe")
# os.environ["webdriver.firefox.bin"] = firefoxBin
dr = webdriver.Firefox()

dr.maximize_window() # 最大化浏览器

dr.get("https://www.baidu.com") # 通过get()方法，打开一个url站点

dr.quit() # 关闭并退出浏览器
