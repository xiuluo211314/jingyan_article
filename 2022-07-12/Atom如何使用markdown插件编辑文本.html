方法/步骤 
1. 下载软件，双击安装，安装完成后打开软件，界面如图所示 
链接: https://pan.baidu.com/s/1rEfktV2TEmxtD35XiwustQ 提取码: emgk
2. 找到右侧的welcome标签页，点击【Install Package】如图
3. 步骤2后，弹出如图所示的布局，点击【Open Installer】
 
4. 或者点击菜单来 Packages >> Settings View >> Open>>Install
 
5. 在下图所示的搜索框中输入： language-markdown 后，点击【Install】进行安装
6. 同理步骤5，输入 markdown-writer 进行搜索和安装
7. 输入markdown-preview 并搜索，会发现默认已经安装
8. 选择菜单栏Packages >> Markdown Preview >> Toggle Preview 可以打开预览窗口
9. 在新建的markdown文件中输入如图所示文字，右侧预览效果如图所示
