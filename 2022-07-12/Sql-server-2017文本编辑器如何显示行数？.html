Sql server 2017文本编辑器如何显示行数？
简介
Sql server 2017文本编辑器如何显示行数？简单几步，让你轻松解决。
经验视频

工具/原料
联网电脑
方法/步骤 
1. 下载安装sql server软件并打开，如图所示
2. 接着，连接到数据库引擎，如图所示，输入服务器名称、登录名和密码。
 

3. 点击菜单栏的【文件】->【新建】->【数据库引擎查询】如图所示
 
4. 点击菜单栏【工具】->【选项】
 
5. 弹出的【选项】窗口中，左侧中找到【所有语言】->【常规】，然后在右侧面板中找到并勾选【行号】
 
6. 会到文本编辑器中，查看代码可以看到已经显示了行号
