python3如何下载安装和使用lxml？?简单几步，让你轻松解决。
经验视频

工具/原料
联网电脑
方法/步骤 
1. 点击电脑的【win】键，弹出的菜单中输入搜索信息：cmd，点击如图所示的【命令提示符】亦可
 


2. 确保安装的python3.*版本，打开命令提示符，输入代码：python --version查看版本
 
3. 命令行中输入： pip3 install lxml==4.1.1 这个版本有etree模块，最后按下回车键进行安装

 
4. 打开IDE创建py脚本，添加代码的： from lxml import etree
 
5.或者登陆网址https://www.lfd.uci.edu/~gohlke/pythonlibs/#lxml 根据自己ptyhon的版本进行下载
6. 下载后使用命令进行安装： pip install C:\Users\admin\Downloads\lxml-4.3.0-cp36-cp36m-win_amd64.whl 注意：最后一个参数指定你在步骤5中下载的whl文件的位置
