简介

方法/步骤 
1. 在 HTML 中，某些字符是预留的，例如<,>等。如果希望正确地显示预留字符，我们必须在 HTML 源代码中使用实体字符（&lt;等）
 
2. 实体字符转换成预留字符。打开VS，创建测试代码
 
3. 输入代码如下：
static void Main(string[] args)
{
string s = "&lt;p&gt;&lt;span&gt;各位领导：&lt;/span&gt;&lt;/p&gt;&lt;p style=\"margin-left:30px;\"&gt;您们好！" +
"target=\"_blank\"&gt;报表查询&lt;/a&gt;）查阅。" +
"&lt;/p&gt;&lt;p style=\"margin-left:30px;\"&gt;祝好！&lt;/p&gt;&lt;p style=\"margin-left:30px;\"&gt;PPC 2019-02-22 09:25:32&lt;/p&gt;";

Console.WriteLine(System.Net.WebUtility.HtmlDecode(s));

}

string s = "You & me > them";
Console.WriteLine(System.Net.WebUtility.HtmlEncode(s));
