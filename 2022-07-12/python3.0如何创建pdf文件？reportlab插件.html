python3.0如何创建pdf文件？reportlab插件?简单几步，让你轻松解决。

经验视频

工具/原料
联网电脑
方法/步骤 
1. 安装好python后，按下win键后，搜索栏中输入cmd弹窗中点击【命令提示符】
 
2. 输入命令并回车，安装插件：pip install reportlab
 
3. 打开python ID（PyCharm）E创建py脚本，或者使用任何的代码编辑器创建
 
4. 弹出的创建文件窗口中，输入文件名称，点击【OK】
 
5. 创建的脚本文件中输入下面的代码
#!/usr/bin/python
from reportlab.pdfgen import canvas
def hello():
c = canvas.Canvas("helloworld.pdf")
c.drawString(100,100,"Hello,World")
c.showPage()
c.save()
hello()
6. 打开创建的py文件，鼠标右键菜单中点击【Run '**.py'】(需要先设置python运行环境)
7. 步骤6中会在当前py文件所在的目录生成pdf文件，双击后打开文件，显示如图所示
