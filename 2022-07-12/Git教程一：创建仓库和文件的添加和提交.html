方法/步骤 
1. 安装好软件Git，例如Git for Windows.exe。安装完成后，在本地磁盘中创建一个文件夹，该文件夹将是自己的仓库的位置和路径。
 
2. 输入命令： git init 来初始化Git仓库，会生成一个隐藏的.git文件夹
 
3. 创建一个文本内容用来测试，并保存
 
4. 步骤3新建文件后（违背追踪），输入命令： git status后回车，显示如图所示的提示
 
5. 输入命令： git add -A 表示的是将所有的文件都添加到仓库中去，即所有的文件都被追踪。输入命令：git status 查看文件已经被追踪了（add成功）
 

6. 命令行中输入：git commit -m "自己的提交信息" ，例如： git commit -m "excute git commit result"，显示结果如图。commit命令是将暂存区的内容提交到本地版本库的当前分支

7. 执行命令： git log ，输出修改信息如图所示，其中包括”谁“修改（添加）了什么内容（txt文本），并提交了什么信息?（excute git commit result----相当于备注）
