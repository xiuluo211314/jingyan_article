window server 2012下如何安装OpenSSH
简介
window server 2012下如何安装OpenSSH?简单几步，让你轻松解决。
经验视频

工具/原料
联网电脑
方法/步骤 
1. 找到官网进行下载（需要梯子），不能下载见步骤2
2. 或者下载 链接: https://pan.baidu.com/s/1d69TH7QF_2up31CTquMTWA 提取码: 673g。如果你是64位的电脑下载64位的
3.将下载的文件解压到文件夹路径（openSSH可以自己在c盘创建，一定解压到这个路径）：C:\Program Files\OpenSSH
4.打开命令提示符，使用cd命令到步骤3中文件夹OpenSSH的位置，然后输入命令后回车： powershell.exe -ExecutionPolicy Bypass -File install-sshd.ps1
 
5. 步骤4成功后，继续如下的命令后回车： netsh advfirewall firewall add rule name=sshd dir=in action=allow protocol=TCP localport=22

6.然后使用命令打开sshd,如下： net start sshd
