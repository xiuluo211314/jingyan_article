创建 Git 仓库的两种方法。简单几步，让你轻松解决。

工具/原料
联网电脑
方法/步骤 
1. 在官网下载git安装软件并进行安装，在开始菜单栏中找到快捷方式如图，点击打开git bash
 
2. 在本地磁盘中创建自己的项目文件夹并进入，如图所示。
 
3. 在步骤1中弹出的窗口中，输入下图所示的命令，定位到你的项目文件夹
 
4. 命令行中输入：git init ，该命令将创建一个名为 .git（默认是隐藏的文件夹） 的子目录，这个子目录含有你初始化的 Git 仓库中所有的必须文件。
 
5. 项目文件夹下创建README.md文件，
6. 命令行中输入下图中的命令并回车：git add README.md
7. 命令行中添加commit命令: git commit -m "init pro"

8. 第二种方法是，直接在项目文件夹下鼠标右键选择【Git bash here】,再执行步骤4-7即可。
