【编程】Eigen库安装教程
简介
Eigen库是开源的矩阵和矢量运算库。如何安装呢？简单几步，让你轻松解决。

经验视频

工具/原料
电脑 联网
方法/步骤 
1.  打开百度浏览器，输入下图所示的内容进行搜索，选择第一个官方网站并打开
2. 步骤1后，打开的网址如下图所示，可以下载最新发布的版本，或者下载其他版本或者源码
3. 在步骤2中解压下载好的文件，并把它放到某个磁盘下（后面会引用此处的路径），子级文件如下图所示。
4. 打开VS，新建VC++控制台应用程序如下图所示，设置名称、路径——>【确定】->【下一步】-【下一步】-【完成】


5. 在解决方案下的源文件鼠标右键-添加-新建项，创建cpp文件如下图所示。

6. 步骤6后输入代码如下
#include <iostream> #include <Eigen/Dense> using namespace Eigen; using namespace std; int main() { MatrixXd m = MatrixXd::Random(3,3); m = (m + MatrixXd::Constant(3,3,1.2)) * 50; cout << "m =" << endl << m << endl; VectorXd v(3); v << 1, 2, 3; cout << "m * v =" << endl << m * v << endl; }
7. 步骤6中出现文件引用不到。选择项目鼠标右键选择属性
8. 步骤7后打开属性页-【VC++目录】-【包含文件目录】-向下的三角号-编辑

9. 选择【...】-找到步骤3中库文件的位置如下图，设置包含路径如下图所示-【选择文件夹】-【确定】
