方法/步骤 
1. 安装Sublime Text 3软件，并安装好插件SublimREPL，参考经验如下：
http://jingyan.baidu.com/article/3aed632ec24a2370108091f4.html
 
2. 在路径C:\Users\admin\AppData\Roaming\Sublime Text 3\Packages\SublimeREPL\config\Python下，找到Main.sublime-menu文件
 
3  搜索【repl_python】如图所示，并修改 保存【cmd】行为"cmd": ["python", "-i", "-u","$file_basename"]
 
4. 点击菜单栏【Preferences】->【Key Bindings】如图所示
5. 添加代码如下并保存，F8键就可以调试了
[
{
"keys": ["f5"],
"caption": "SublimeREPL: Python - RUN current file",
"command": "run_existing_window_command",
"args": {
"id": "repl_python_run",
"file": "config/Python/Main.sublime-menu"}
},
{
"keys": ["f8"],
"caption": "SublimeREPL: Python - PDB current file",
"command": "run_existing_window_command",
"args": {
"id": "repl_python_pdb",
"file": "config/Python/Main.sublime-menu"}
},
]


 

6. 每次F5调试都会新建窗口比较烦，因此找到步骤3文件的【repl_python_run】，并添加代码保存："view_id": "*REPL* [python]",
7. 找到 C:\Users\admin\AppData\Roaming\Sublime Text 3\Packages\SublimeREPL下sublimerepl.py文件并打开
8. 编辑：if view.id() == view_id: 替换为 if view.name() == view_id: 如图所示(注意：最好直接将id改为name,不要复制粘贴)
