工具/原料
联网电脑
方法/步骤 
1.  登录谷歌yun平台，选择【设置】-【Compute Engine】-【VM 实例】，创建实例参考经验如下：
2. 创建好实例后，点击如图所示的【连接】->【在浏览器窗口中打开】（弹出窗口如果被阻止，请再次尝试）
3. 步骤2后，弹出下图所示的命令行窗口，输入命令： sudo -i
4. 输入命令： wget -N --no-check-certificate https://raw.githubusercontent.com/FunctionClub/YankeeBBR/master/bbr.sh && bash bbr.sh install
出现如图的蓝底窗口按TAB键选NO

5. 选择重启 Y。

6. 步骤5后，按下任意键，弹出下图所示的内容，点击【重新连接】

7. 步骤6后，输入命令： sudo -i ，再回车

8. 输入命令: bash bbr.sh start

9. 输入命令：wget --no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocksR.sh && chmod +x shadowsocksR.sh ，再回车

10. =输入命令：./shadowsocksR.sh ，再回车
11. 输入密码，再回车
12. 输入端口号，再回车

13. 接下默认，按下回车键
14. 等待默认重启，账户和密码结果如图，参考如下经验进行连接
