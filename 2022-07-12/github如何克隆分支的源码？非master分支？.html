方法/步骤 
1.github的源码有的有多个分支，如图所示master分支和1分支
 

2.默认git下载到的源码是maste分支的，所以我们需要进行切换分支
 
3.cd命令切换到保存源码的路径，如图所示
 
4.使用git clone...命令进行下载，如图所示
 
5.步骤4后，定位到下载后的文件夹，使用ls -a查看所有文件，此时是master分支上的文件
6.git branch
git checkout -b 1 origin/1 其中1是分支的名称
