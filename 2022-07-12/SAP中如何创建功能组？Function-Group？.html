方法/步骤 
1. 打开sap软件，在下图所示的位置输入事务码：se37 ,按下回车键进入
 
2. 首先，新建【Function Group】.点击menu(菜单栏)->goto-> function groups->create group
3. 弹窗中输入【function group】、 【short text】（描述），然后点击【save】
 
4. 弹出的【create object directory entry】中输入包名，点击左下角的【Local Object】
 
5. 输入事务码: /NSE80 回到如图所示的界面
6. 步骤5后，输入ZHH_TEST_FUNGROUP(之前步骤创建的)，查看功能组的文件信息如图所示
7. 右键菜单中激活该功能组，如图
