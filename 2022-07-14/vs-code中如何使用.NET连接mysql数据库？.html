方法/步骤 
1. 安装好软件后，打开界面显示如图所示
 
 
2. 准备好MySql数据库，并创建好需要的信息，在代码中需要使用相关的连接串
 
3. 在.NET项目中，创建相关的数据库连接信息和代码
4..NET连接MySQL数据库要引用MySql.Data.dll动态库，按下快捷键ctrl+~后，会在底部打开终端面板，如图所示的MsSqlConnection是引用不到报错


5.在终端输入命令后回车：dotnet add package MySql.Data，vscode会自动下载并生成相应的MySql.Data.dll
6. 然后添加使用引用代码：using MySql.Data.MySqlClient;,就能连接MySQL数据库。
