code-server 如何安装vs code在线编辑器？?简单几步，让你轻松解决。

工具/原料
系统版本：Windows 10 专业版;21H1
品牌型号：Windows 10 专业版;21H1型号
软件版本：Win10;10
其他：无;无1
方法/步骤 
1. 使用浏览器打开github的网址搜索code-server的仓库位置，找到release下面，如图所示发布的版本
 
2. 根据自己的机器的cpu架构，选择适当的安装包，示例： code-server-3.4.1-linux-x86_64.tar.gz
或者code-server-4.0.0-linux-amd64.tar.gz
 
3. 网络便捷的话可以使用命令：wget进行下载，如图所示。
 
4. 或者现在电脑端下载完成，再使用scp的命令网络传输到服务器，命令格式如下：
scp 下载的压缩包文件路径 服务器用户名@服务器ip:服务器上传的文件夹路径
 
5. 使用命令进行解压： tar zxvf ./code-server-3.4.1-linux-x86_64.tar.gz
6. 使用cd 命令进入到解压后的安装包文件夹下，再进入到bin文件夹下
7. 执行命令： ./code-server 后回车，显示如下说明启动成功了。但是目前只能本机访问
8.使用命令进入到配置文件路径下： cd ~/.config/code-server/
9.使用vi命令打开config.yaml配置文件，显示如图所示

10. 第一行可以修改绑定的ip和端口号，0.0.0.0:8889表示任何ip都可以访问8889端口；password表示密码需要记录下来；使用pwd命令找到code-server解压包的bin文件夹下

11. 进入到bin目录下，直接执行命令： ./code-server后回车，这样已经运行成功如图所示。打开浏览器访问ip:8890后回车，可以打开网页如图所示，输入步骤9中记录的密码后点击提交submit按钮


12. 步骤11登录后，显示的网页版本的vs code 如图所示
