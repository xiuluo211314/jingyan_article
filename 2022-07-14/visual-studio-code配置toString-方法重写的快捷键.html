方法/步骤 
1. 下载vscode软件，并安装后打开，如图所示
 
2. 点击顶部菜单栏：File--> Preferences--> Keyboard Shortcuts
 
3. 步骤2后，在打开的页面中，点击右上角如图所示的文件图标
 
4. 在弹出的json配置文件,显示如图所示，我们接下来需要配置该文件
 
5. 在步骤4中的json配置文件中，添加如下代码，如图
{
"key": "shift+alt+g", //g for generate
"command": "editor.action.codeAction",
"args": {
"kind": "source.generate", //or "source.generate.accessors" to narrow it down
}
}
 

6. 最后，在java代码新建的类文件中，空白处右键菜单中，显示如图所示，点击第二个Generate toString(),并在图示的弹窗中，选择OK

7. 在java类文件中，显示重写的toString()方法如图所示
