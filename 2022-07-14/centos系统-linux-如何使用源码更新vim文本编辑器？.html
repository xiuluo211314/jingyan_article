方法/步骤 
1.打开命令行后，执行命令： sudo yum -y remove vim

2. 再输入命令行后回车：yum install gcc make ncurses ncurses-devel
 
3. 在执行的过程中，出现如图所示的Is this ok ?[y/d/N]: ，这个时候输入按键Y后回车，等待complete即是完成
4. 输入命令克隆vim的原码： sudo git clone https://github.com/vim/vim.git

5. 等待步骤4中代码下载完成后，进入操vim的文件夹下，命令如下：cd ./vim/
 
6. 执行命令后回车： make， 如图所示进行编译

 


7. 步骤6中编译完成后，使用命令进行安装： sudo make install
