方法/步骤 
1. 注册并登陆个人账户，进入到个人的页面。新建一个仓库，例如如图所示parent

 
2. 点击如图所示的【克隆/下载】按钮，复制链接url
 
3. git版本控制工具切换远程仓库地址：git remote set-url origin url ，其中url是步骤2中复制的内容


 
4. git pull origin master
5. 在打开的bash终端中，输入命令： git add . 添加所有文件到git追踪关系中
6. 使用commit命令提交变更的内容：git commit -m ‘提交的信息’

7. 使用命令拉取远端的仓库数据到本地： git pull origin master --allow-unrelated-histories

8. 提交变更到远端： git push -u origin master
