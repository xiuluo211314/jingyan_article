CentOs7中如何命令行安装最新版本的git？?简单几步，让你轻松解决。
经验视频

工具/原料
联网电脑
方法/步骤 
1. ssh登录服务器或者打开电脑的终端
2. 输入git命令后回车，报错如图所示，说明没有安装git
-bash: git: command not found
 
3. 使用yum安装的git版本比较低，因此本教程采用编译安装的方式安装最新版本。使用如图中的命令下载安装包。或者使用win现在后上传到centos
4. 下载完成后使用ls命令查看安装包的所在文职
 
5. 使用命令进行解压： sudo tar xzvf git-2.9.2.tar.gz

 
6. 给文件夹设置读写权限： sudo chmod 777 ./git-2.9.2
7.进入到解压后的文件夹 cd git-2.9.2

8. 使用 make configure命令后，报错显示如下：
/bin/sh: autoconf: command not found
make: *** [configure] Error 127

9. 遇到步骤8中错误需要使用命令安装，安装过程中选择y（yes）： sudo yum install autoconf
 
10. 使用命令： make configure
11. 进入root权限后使用命令安装， yum install zlib-devel
yum install perl-ExtUtils-CBuilder perl-ExtUtils-MakeMaker
yum install curl-devel expat-devel gettext-devel openssl-devel zlib-develperl-devel

12. 使用命令： make profix=/usr/git
13. 使用命令安装： make install

14. 打开文件：vim /etc/profile
15. 配置环境变量 :/usr/git/bin
PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin:/usr/git/bin

16. 更新环境变量信息： source /etc/profile
17. git --version 查看安装是否成功
