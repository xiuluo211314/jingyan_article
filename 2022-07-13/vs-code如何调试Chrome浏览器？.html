方法/步骤
1. 下载并安装vs code.

2. 点击界面左侧的拓展按钮，在如图所示的搜索中搜索插件： chrome debu。。。找到如图所示的插件，在右侧的详情页面中点击install按钮。

3. 安装完成后，找到如图左侧栏中的调试按钮，再店家如图所示的设置按钮，右侧的面板中下拉框中选择chrome

4. 在打开的lanuch.json文件中，输入下图中的文本并保存
"type": "chrome",
"request": "launch",
"name": "vuejs: chrome",
"url": "http://localhost:8083",
"webRoot": "${workspaceFolder}/src",
"breakOnLoad": true,
"sourceMapPathOverrides": {
"webpack:///./src/*": "${webRoot}/*"
}

5. 在命令行中输入命令运行程序： npm run dev

6. 按下键盘的F12后，找到Sources下的代码，打下断点，运行后显示调试效果如图所示。
