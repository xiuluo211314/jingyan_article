方法/步骤 
1.使用IDE打开springboot的代码工程
 
2. 找到项目启动器同文件夹下的config文件夹，如果没有则新建
 
3.创建java文件用于后面的测试
 
4. 在步骤3中创建的java文件中写入如图所示的代码。注意注解是需要的
 
5. 创建一个util.java的帮助工具类，在里面定义静态变量如下，并创建getter和setter静态方法。同时在yml文件中创建需要使用的变量，如图。

6. 回到步骤4汇总创建的文件中，写下如下代码。最后创建测试文件进行即可。
@Configuration
public class StaticConfig {
@Value("${attachmentPrefixUrl}")
private String attachmentPrefixUrl;

@Bean
public int initStatic(){
PdfUtils.setAttachmentPrefixUrl(attachmentPrefixUrl);
return 0;
}
}
