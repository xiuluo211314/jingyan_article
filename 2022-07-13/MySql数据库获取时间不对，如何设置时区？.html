方法/步骤 
1. 使用终端进入到linux操作系统，输入命令date -R 如图所示查看系统当前日期和时区。如图所示的时区是北京时间时区是符合条件的
 
2. 接下来查看mysql中的时区是否和步骤1中一致。使用客户端链接mysql，打卡命令查询窗口，输入命令: show variables like '%time_zone%'
 
3.步骤2中的命令执行后，查询结果如图所示，说明是不一致的
 
4. 执行命令：set global time_zone = '+8:00'

 
5. 执行命令后即可： set time_zone = '+8:00
6. 执行保存更改命令：flush privileges
