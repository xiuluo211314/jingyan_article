方法/步骤 
1. Vue是一套构建用户界面的渐进式框架, Vue-cli是快速创建vue工程的脚手架，非常方便
2. 使用vs code创建项目，打开控制台输入命令： npm uninstall vue-cli -g
 
 
3. Vue CLI 需要 Node.js 8.9 或更高版本 (推荐 8.11.0+)，使用命令查看node版本： node -v
4. 使用命令安装：npm install -g @vue/cli
 
5. 安装完成后，使用命令查看版本： vue --version
6. 创建hello-world项目： vue create hello-world
