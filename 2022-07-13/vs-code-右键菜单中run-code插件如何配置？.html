工具/原料
联网电脑
方法/步骤 
1.打开vs code并创建java工程。
 
2.点击左侧工具栏的扩展，搜索 Code Runner插件并安装
3.安装完成后，点击左下角的设置按钮，弹出的菜单中选择Settings
 
4.在打开的页面中搜索run code，找到run code configuration.这里推荐勾选Clear Previous Output
 
5. 找到Executor Map选项，如图所示，点击edit in settings.json
6. 步骤5后后跳转并打开json文件，复制下面的代码并拷贝到如图所示的区域即可，最后保存关闭json文件：
"java": "cd $dir && javac $fileName && java $fileNameWithoutExt",
