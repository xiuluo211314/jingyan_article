方法/步骤 
1. 打开vs code软件如图所示
2. 点击左侧菜单栏的拓展，并搜索插件安装： Maven for Java
 
3. 插件安装完成后，点击左下角的设置按钮，弹出的菜单中选中settings
4. 在跳转的页面的搜索框中输入maven后回车搜索，找到如图所示的java选项，探后点击右侧的edit in settings.json 如图所示
5. 找到本地maven配置的settings的全路径，复制粘贴到图示的位置保存
6. 左侧找到Maven for java选项，然后在打开的页面中找到如图所示的Maven>Executable Path。注意此处输入maven的安装路径，指向mvn所在的文件夹路径。

7.重启vs code软件后，默认maven插件会自动读取并新建如图所示的文件夹。用户可以自己选中创建好的pom文件并在右键菜单中选择Update project configuration，手动更新，如图所示。
