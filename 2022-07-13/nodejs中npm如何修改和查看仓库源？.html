方法/步骤 
1. 下载并安装nodejs和npm
 
2. 安装配置完成后，打开命令行工具，输入命令查看现有的仓库源： npm config get registry
 
3. 步骤2回车后，显示结果如图所示，即是npm当前的仓库源的地址
 
4. 接着输入设置仓库源的命令： npm config set registry https://registry.npm.**.org。详细命令请看截图。
5. 安装插件的命令使用npm install ***即可进行安装。
6. 再次使用命令查看替换后的仓库源结果如下：npm config get registry
