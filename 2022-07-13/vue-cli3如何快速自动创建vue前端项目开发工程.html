方法/步骤 
1. 参考经验升级vue-cli2到3,或者自己安装最新版本：http://jingyan.baidu.com/article/0a52e3f46fffdbff62ed72da.html
 
2. 创建好项目后，打开控制台，输入命令如下：vue create vue-zhh-worl-tool-2
 
3.步骤2后，出现如图的选择界面，使用方向键进行选择，选择如图所示的 Manually select features 后，按下回车键
4. 后续步骤中勾选如图所示的内筒
(*) Router
(*) Vuex
(*) CSS Pre-processors
( ) Linter / Formatter
(*) Unit Testing
5. 进入到如图所示的指导页面，按下回车键进行自动搭建
6. 出现如图所示的命令，第一个是定位到项目的文件夹路径下，第二个是运行项目的命令
7. 运行命令： npm run serve， 结果显示如图所示，表示运行成功。打开浏览器输入http://localhost:8080/ 查看网页即可
