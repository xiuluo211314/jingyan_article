方法/步骤

使用vs code打开vue前端工程，并打开终端，如图所示
 
2. 输入命令后回车： npm i element-ui -S

 
3. 步骤2中安装完成，如图所示
4. 找到main.js文件，加入如图所示的引用代码：
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
Vue.use(ElementUI)
 
5. 在终端中输入命令： npm run dev

6. 浏览器中输入url回车：http://localhost:8080， 显示如图所示
