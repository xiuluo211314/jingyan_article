centos7服务器上如何安装python3
简介
centos7服务器上如何安装python3?简单几步，让你轻松解决。
经验视频

工具/原料
联网电脑
方法/步骤 
1. 打开远程工具，输入命令查看服务器中python的安装版本： ls /usr/bin/python*
 
2. 使用命令下载python3的安装文件：wget https://www.python.org/ftp/python/3.6.2/Python-3.6.2.tar.xz
 
3.进入root权限，使用命令创建文件夹用于安装解压步骤2中现在好的文件： mkdir /usr/local/python3
 
4. 解压下载后的文件： tar -xvJf Python-3.6.2.tar.xz

5. 安装相关的依赖：
yum -y groupinstall "Development tools" yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel
6. 依次执行以下命令：
cd Python-3.6.2
./configure --prefix=/usr/local/python3
make && make install
7. 使用命令创建软连接：
ln -s /usr/local/python3/bin/python3 /usr/bin/python3 ln -s /usr/local/python3/bin/pip3 /usr/bin/pip3
 
8. 输入python3后回车，显示如下，说明安装成功。
