方法/步骤 
1. 登录服务器，并获取root权限
 
2. 使用apt-get update，进行更新
 
3.使用命令安装redis: apt-get -y install redis-server
4. 输入命令并回车确认redis已经运行: ps -f -u redis。redis安装成功，但是只接受来自127.0.0.1连接到redis.

 
5. 输入命令： redis-server 后回车。显示如图所说明安装成功
6. 输入命令redis-cli进入交互环境
