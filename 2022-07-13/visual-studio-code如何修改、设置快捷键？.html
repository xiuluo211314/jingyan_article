方法/步骤 
1.打开vscode项目，如图所示是java项目。黄灯显示的快捷键是ctrl+.，但是没有作用
 
2. 找到菜单栏File下的preferences-->keybord shortcuts
 
3. 弹出如图所示的所有快捷键的配置面板
 
4. 在搜索框中输入quickfix后，找到如图所示的修改按钮并点击
 
5. 弹出弹窗如图所示，这是可以操作键盘中按键，设置自己需要的快键键组合
6. 设置完成后，回到编辑区域后，按下步骤5中的快捷键后，会弹出如图所示的提示菜单，自行选择即可。
