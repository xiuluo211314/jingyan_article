3. 弹出的命令行工具显示如图所示，会自动定位到当前文件夹的路径下
 
4. 使用命令设置新的远程url： git remote set-url origin 新的仓库地址
 
5. 弹出如图所示的OpenSSH窗口，属入用户名和密码
6. 步骤5后，点击如图所示的OK按钮。
