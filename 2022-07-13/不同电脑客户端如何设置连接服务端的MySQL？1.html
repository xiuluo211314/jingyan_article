工具/原料

方法/步骤 
1. 使用远程工具连接自己的服务器，确保开启了mysql服务
 
2.参考经验检查阿里云服务器的安全组策略，添加3306 端口的入口，
https://jingyan.baidu.com/article/17bd8e52654f4985ab2bb8c6.html
 
3.在命令行窗口中输入mysql -u root -p
4.输入命令并回车: use mysql;
 
5. 输入命令并回车：update user set host = '%' where user = 'root';
6. 输入命令如下并回车： select user, authentication_string from user;
7.如果出现错误：ERROR 1130: Host '192.168.1.3' is not allowed to connect to thisMySQL serve，请继续参考经验：不同电脑客户端如何设置连接服务端的MySQL？2
