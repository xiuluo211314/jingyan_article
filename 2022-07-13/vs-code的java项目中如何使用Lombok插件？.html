工具/原料

方法/步骤 
1.打开vs code软件
 
2. 点击左侧工具栏的拓展按钮，如图所示，弹出的搜索面板中输入关键词进行搜索： Lombok Annotations Support for VS Code
 
3. 找到步骤2中的插件后，点击如图所示的install按钮，安装插件
 
4. 安装完成后，重启软件，然后参考经验创建maven项目
https://jingyan.baidu.com/article/f25ef2540177dd092c1b82bf.html
5. 双击打开如图所示的pom.xml文件
6. 在pom文件中加入lombok插件的依赖，后续更新pom文件使用update project configuration即可。
<dependency>
<groupId>org.projectlombok</groupId>
<artifactId>lombok</artifactId>
<version>1.18.12</version>
<scope>provided</scope>
</dependency>

7. 右下角弹出pom文件更改后自动更新，选择always
 
8. 创建测试代码如图所示，运行java程序后，输出结果如图。
