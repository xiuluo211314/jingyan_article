工具/原料
联网电脑
方法/步骤 
1. IDEA中项目右键菜单中-->webservice--> Generate Java Code From Wsdl中显示界面不是如下图所示，需要执行后续步骤。否则直接使用。
2. 鼠标点击项目菜单的File,弹出的菜单中点击【Project Struncture...】
3. 在弹出的界面中选择Modules，右侧界面中选择额如图所示的webservices Client
4.在selecct WS Engine下拉框中选中如图所示的选项
 
5.提示需要安装插件如图所示，点击yes
6. 弹出需要重启选择restart
7. 重启后，点击项目右键菜单中选择【generate java code from wsdl】
8. 弹窗中填入如图所示的信息。其中wsdl.wsdl文件是url中显示的xml内容保存文件到本地，并修改文件扩展名称为**.wsdl
 
9. 步骤7后，生成的代码如图所示
