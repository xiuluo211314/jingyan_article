方法/步骤 
1.使用vs code配置java开发环境的时候，运行java命令出现错误：找不到或无法加载主类的问题。首先，使用javac命令确认java环境安装成功，并且编译成功，如图
 
2. 在vs code的命令行工具中，使用java testMain 命令，运行编译好的程序报错如图所示。

3. 经原因排查后，发现是引用package包的问题。代码中加入了报名，编译后的文件中也生成了包名，但运行的时候找不到主类文件
solutions.src.com.zhh.leetcode
 
4. 在命令行中输入命令并回车： javac -d. .\testMain.java
5. 再次执行如下命令，运行结果如图所示。
java solutions.src.com.zhh.leetcode.testMain
6. 步骤4命令后，会生成如图所示的存放编译后的文件夹的文件，用于运行。
