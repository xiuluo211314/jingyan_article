工具/原料

方法/步骤 
1. 下载安装redis桌面管理软件，如图图所示
 
2. 点击底部connect to redis server
 
3.弹出的连接窗口中，查看选项卡
4. 找到SSH Tunnel选项卡，输入ip 端口号以及用户名和密码（或者私钥的文件路径）。此步骤需要在服务 器中安装配置好redis或者本机安装配置好redis
 
5. 点击如图的Test Connection， 弹出成功窗口如图所示
6. 连接成功后，进入到如图所示的redis数据库中，至此连接成功
