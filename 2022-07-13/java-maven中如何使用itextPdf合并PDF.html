方法/步骤 
1.创建sprintboot项目
2. 创建PdfUtils.java文件，用于后面写代码
3. 找到项目的pom.xml文件，加入下面的依赖
<dependency>
<groupId>com.itextpdf</groupId>
<artifactId>itextpdf</artifactId>
<version>5.4.2</version>
</dependency>
 
4. 步骤3后会自动下载相关的依赖文件，出现右下角的导入问题，选择开启自动导入功能。
 
5. 在代码文件中，写入下面的方法：
public static boolean mergePdfWithPaths(List<String> files, String newfile){
Document document = null;
boolean isSuccess = false;
try {
document = new Document(new PdfReader(files.get(0)).getPageSize(1));
PdfCopy copy = new PdfCopy(document, new FileOutputStream(newfile));
document.open();
for (int i = 0; i < files.size(); i++) {
PdfReader reader = new PdfReader(files.get(i));
int n = reader.getNumberOfPages();
for (int j = 1; j <= n; j++) {
document.newPage();
PdfImportedPage page = copy.getImportedPage(reader, j);
copy.addPage(page);
}
}
isSuccess = true;
} catch (IOException e) {
e.printStackTrace();
} catch (BadPdfFormatException e) {
e.printStackTrace();
} catch (DocumentException e) {
e.printStackTrace();
}finally {
if(document != null){
document.close();
}
return isSuccess;
}
}
6. 创建测试方法，代码如下：
void mergePdfFiles() {

List<String> files = new ArrayList<>();
files.add(pdf1);
files.add(pdf2);
files.add(pdf3);
boolean res = PdfUtils.mergePdfWithPaths(files, pdfRes);
Assert.assertTrue(res);
}
